🎨 Pleasant Cove Widget Test

This page demonstrates the embeddable chat widget. Look for the floating chat button in the bottom-right corner. Click it to open the messaging panel.

Embed Code

<script src="https://ararrbvhzaudfaxjwdrc.functions.supabase.co/functions/v1/widget" data-project-token="test-acme-plumbing-2024" data-theme="light" data-position="bottom-right" data-accent="#6366f1" defer ></script>

Configuration Options

data-project-token — Required. The unique token for this project.

data-theme — "light" or "dark" (default: light)

data-position — "bottom-right", "bottom-left", "top-right", "top-left"

data-accent — Hex color for button and accent (default: #111827)

💡 The widget polls for new messages every 8 seconds while the panel is open. Messages you send will appear in the admin dashboard instantly.
